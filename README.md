# silly_diff Demo App
Take a try: http://diffdemo.azurewebsites.net/

Allows to compare XML 2 XML from very simple UI.
Demonstrates basic functionality of Silly_Diff library.

## Requirements:
* Silly_Diff
* jsdifflib
* Grails, Gradle, Bootstrap, jQuery, AngularJs
